# Сводка исследования: Ретроспективный анализ пандемии COVID-19 на уровне отдельного города (2020–2025 гг.)

## Общие сведения

| Параметр | Значение |
|----------|----------|
| **Объект** | г. Псков (~200 000 жителей) |
| **Период** | 17 марта 2020 — 30 декабря 2024 (1750 дней) |
| **Тип исследования** | Ретроспективное описательное с элементами математического моделирования |
| **Источники данных** | Формы ВП (внебольничная пневмония), оперативные сводки COVID-19 Роспотребнадзора по Псковской области |
| **Авторы** | Самаркин А.И., Самаркина Е.И., Ревенко А.Б., Щеткина, Иванова Н.В., Белов В.С. |

---

## Цель исследования

Двойная цель:

1. **Методологическая** — разработка воспроизводимого аналитического конвейера ретроспективного анализа пандемии на основе данных рутинного санитарно-эпидемиологического надзора, применимого к малым и средним городам.
2. **Содержательная** — проверка гипотезы о том, что порядок дробной производной α_k является информативным параметром, отражающим нарастание популяционного иммунного фона в ходе последовательной смены волн.

---

## Структура исследования

### Введение (раздел 1)

**Научный контекст (1.1):** Пандемия COVID-19 обнажила ограничения стандартных компартментных моделей с производными целого порядка при работе с реальными данными рутинного надзора. Дробное исчисление (производная Капуто) предоставляет математически строгий инструмент для включения наследственных эффектов («памяти» системы) в динамические модели.

**Обзор литературы (1.2):** Систематизированы работы по четырём направлениям:
- Классическая теория компартментных моделей (Kermack–McKendrick, Anderson–May, SEIR/SEIRD)
- Биологические параметры вариантов SARS-CoV-2 (Xu et al., Madewell et al. — метаанализы инкубационного периода, серийного интервала)
- Дробное исчисление (Подлубный, Дитхельм, схема Грюнвальда–Летникова, метод Адамса–Башфорта–Мултона)
- Вычислительный инструментарий (DifferentialEquations.jl, язык Julia)

**Исследовательский зазор (1.3):** Выявлены четыре лакуны:
- Дефицит субнационального масштаба (малые города России)
- Разрыв между биологическими параметрами штаммов и калибровкой волн
- Отсутствие формальной селекции дробных моделей и биологической интерпретации α_k
- Неразвитость воспроизводимых конвейеров для рутинных данных

### Материалы и методы (раздел 2)

**Предобработка данных (2.1–2.2):**
- Исходные данные: Excel-файлы с многоуровневыми объединёнными заголовками
- Восстановление непрерывного ряда: натуральные кубические сплайны с монотонизацией
- Сглаживание: скользящее среднее (окно 18 дней) и гауссово ядро (σ = 15 дней)

**Идентификация волн (2.3):**
- Алгоритм на основе знака сглаженной первой производной
- Автомат: состояния `:up` / `:down`, объединение «подъём + спуск» в одну волну
- Характеристики: пик, продолжительность, AUC, FWHM, асимметрия, выступ пика

**Календарь штаммов (2.4):**
- 10 вариантов: Wuhan → Alpha → Delta → Omicron BA.1/BA.2 → BA.4/5 → XBB → JN.1 → KP.3.1.1/XEC/LP.8.1
- Параметры σ, γ, μ, β, R₀ из метаанализов [4, 5]
- Разрешение перекрытий: приоритет варианту с более поздним началом доминирования

**Модели (2.5–2.6):**

| Параметр | Целочисленная SEIRD | Дробная SEIRD |
|----------|---------------------|---------------|
| Порядок | 1 (ОДУ) | α_k ∈ (0.5, 1.0] (Капуто) |
| Свободные параметры | β, μ, E₀, I₀, N_eff (5) | β, μ, E₀, I₀, N_eff, α_k (6) |
| Фиксированные | σ, γ (из календаря штаммов) | σ, γ (из календаря штаммов) |
| Интегратор | AutoVern7(Rodas5P) | Схема Грюнвальда–Летникова (GL) |
| Функция потерь | log-MSE по случаям (+ log-MSE по смертям при Σd > 5) | То же |

**Оптимизация (2.7):**
- Трёхступенчатая: аналитическое предобусловливание → сеточный поиск (400 узлов) → мультистарт NelderMead (15 запусков) → L-BFGS
- Логарифмическая параметризация всех параметров
- Вес смертности w_d = 3.0 при Σd > 5

**Сравнение моделей (2.8):**
- Метрики: RMSE, MAE, MAPE, R², log-MSE
- Информационные критерии: AIC, BIC
- Тест отношения правдоподобия (LRT): Λ ~ χ²(1), порог p < 0.05, поправка Бонферрони p_adj < 0.005
- Анализ остатков: ACF до 14 лагов, Q-Q-графики

## Научная новизна

1. Впервые для российского малого города проведён систематический поочередной анализ пандемии COVID-19 с формальными критериями выбора между целочисленной и дробной SEIRD-моделями за полный период 2020–2025 гг.
2. Впервые α_k предложен и верифицирован как косвенный количественный маркер нарастания популяционного иммунитета, оцениваемый из данных рутинного надзора без иммунологических обследований.
3. Разработан воспроизводимый аналитический конвейер на языке Julia для работы с разнородными файлами государственной эпидемиологической отчётности.

---

## Ключевые выводы

- **Дробная SEIRD-модель статистически значимо превосходит целочисленную** для 5 из 10 волн (Alpha, Omicron BA.4/5, XBB.1.16, JN.1, KP.3.1.1/XEC/LP.8.1)
- **α_k < 1 интерпретируется как маркер субдиффузионного режима**, обусловленного популяционной иммунной памятью
- **Ранние волны (2020 г.) не требуют дробной модели** — население полностью восприимчиво, марковская кинетика адекватна
- **N_eff монотонно снижается** от ранних волн к поздним, отражая сужение пула регистрируемых восприимчивых
- **R₀_fit значительно ниже R₀_prior** для поздних волн — проявление накопленного гибридного иммунитета
